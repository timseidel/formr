# Disconnect from formr

Disconnects from formr if connected.

## Usage

``` r
formr_disconnect(host = formr_last_host())
```

## Arguments

- host:

  defaults to
  [`formr_last_host()`](http://rubenarslan.github.io/formr/reference/formr_last_host.md),
  which defaults to https://formr.org

## Examples

``` r
if (FALSE) { # \dontrun{
formr_disconnect()
} # }
```
