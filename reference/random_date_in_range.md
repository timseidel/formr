# Random date in range

taken from Dirk Eddelbuettel's answer here
http://stackoverflow.com/a/14721124/263054

## Usage

``` r
random_date_in_range(N, lower = "2012/01/01", upper = "2012/12/31")
```

## Arguments

- N:

  desired number of random dates

- lower:

  lower limit

- upper:

  upper limit
